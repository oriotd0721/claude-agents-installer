📋 Installation & utilisation
  1. Copiez la commande ci-dessus.
  2. Ouvrez PowerShell à la racine de votre projet (le dossier où vous voulez installer les agents/skills).
  3. Collez et exécutez la commande — elle clone/actualise la librairie, installe les agents/skills/hooks sélectionnés dans .claude/, et génère CLAUDE.md + le workspace VS Code.
  4. Ouvrez le fichier .code-workspace généré dans VS Code (ou lancez claude directement dans le dossier).
  5. Tapez /help dans Claude Code pour vérifier que les agents/skills sont bien détectés.
